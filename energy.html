Energy Trade Report

Building upon the 2022 Energy Trade Report, which primarily utilized static bar graphs for trend analysis, this project presents an interactive dashboard for the 2023 data. Using more interactive visualisations, it is easier to identify patterns, and gain actionable insights relevant to national import and export activities.

Project Objective

Source data for imports and exports from SARS website. Analyse the data and visualise it.
